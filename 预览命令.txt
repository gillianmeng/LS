本地预览（复制到终端执行即可）
============================

【首次】创建虚拟环境并装依赖：

cd /Users/mengfanzhuo/Documents/LS
python3 -m venv venv
source venv/bin/activate
pip install -r requirements.txt
python manage.py migrate

【每次】启动：

cd /Users/mengfanzhuo/Documents/LS
source venv/bin/activate
python manage.py runserver

或用脚本（同上，会 migrate + runserver）：

cd /Users/mengfanzhuo/Documents/LS
chmod +x run_preview.sh
./run_preview.sh

浏览器打开：http://127.0.0.1:8000/

---

一行版（已建好 venv 时）：

cd /Users/mengfanzhuo/Documents/LS && source venv/bin/activate && python manage.py migrate && python manage.py runserver

---

说明：本地看图/视频请保持 .env 里 USE_OSS_MEDIA=0（或不要设 USE_OSS_MEDIA），否则地址会指向阿里云 OSS。
